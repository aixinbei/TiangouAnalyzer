# 舔狗分析器

看看你和你朋友之间，到底谁更像舔狗。

这是一个只做微信的本地聊天娱乐分析器。它会读取你当前电脑上、当前登录微信的本地聊天记录，看看你们俩到底是谁更主动、谁更上头、谁更放不下。

## 这版特点

- 只支持微信
- 只支持 Windows
- 只在本机运行
- 不需要云端服务

## 能做什么

1. 自动查找本机微信聊天记录目录
2. 自动解析当前登录微信的数据
3. 加载联系人列表
4. 分析某个联系人里谁更常主动发起对话
5. 查看每一轮对话的完整消息

## 使用方法

### 开发环境启动

第一次先装依赖：

```powershell
install_deps.bat
```

启动：

```powershell
start_app.bat
```

### 使用顺序

1. 先打开并登录电脑版微信
2. 打开本程序
3. 点击“解析本地聊天记录”
4. 等联系人列表出来
5. 选择联系人查看结果

## 注意

- 本程序不联网
- 需要当前电脑上的微信保持登录状态
- 仅供学习和娱乐使用
- 仅限解密和分析本人设备上的本人数据
- 请勿用于任何非法用途
